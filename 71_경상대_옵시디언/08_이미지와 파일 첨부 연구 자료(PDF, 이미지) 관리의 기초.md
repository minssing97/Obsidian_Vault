## 1. 기본 설정: 첨부 파일 관리를 위한 첫걸음
- 옵시디언은 '볼트(Vault)'라는 특정 폴더를 기반으로 작동합니다. 노트에 첨부하는 모든 파일(이미지, PDF 등)은 이 볼트 폴더 내에 저장됩니다. 이는 모든 자료가 내 컴퓨터에 직접 저장되어 안전하고, 다른 프로그램으로 파일을 쉽게 옮기거나 관리할 수 있음을 의미합니다.
- `설정 > 파일 및 링크 > 새 첨부 파일 기본 위치` 메뉴에서 첨부 파일을 저장할 기본 폴더를 지정할 수 있습니다. 예를 들어 'attachments'라는 폴더를 만들어 모든 첨부 파일을 한곳에 모아두면, 노트 파일과 첨부 파일이 섞이지 않아 볼트를 깔끔하게 유지할 수 있습니다.

## 2. 파일을 첨부하는 두 가지 방법 : 링크와 임베딩
옵시디언 노트에 파일을 첨부하는 방법은 크게 '링크'와 '임베딩' 두 가지가 있습니다. 두 방식은 파일을 노트에 어떻게 보여줄지 결정하는 중요한 차이가 있으며, 목적에 따라 적절한 방법을 선택하는 것이 좋습니다.

- **링크(Link)**: `[[파일이름.pdf]]`처럼 파일을 연결하면, 클릭 시 해당 파일을 옵시디언의 새 탭이나 연결된 프로그램에서 열 수 있는 하이퍼링크가 생성됩니다. 노트 본문을 깔끔하게 유지할 수 있습니다. 참고 자료로 첨부하지만, 지금 당장 내용을 볼 필요는 없을 때 유용합니다. 하지만 내용을 확인하려면 반드시 클릭해야 하는 번거로움이 있습니다.

- **임베딩(Embedding)**: `![[파일이름.png]]`처럼 링크 앞에 느낌표(!)를 붙이면, 이미지나 PDF의 내용이 노트 본문에 직접 나타납니다. 노트의 맥락 속에서 이미지나 PDF 페이지를 바로 확인할 수 있어 정보의 흐름이 끊기지 않습니다. 시각 자료를 보며 글을 작성할 때 매우 효과적입니다. 반면, 여러 개의 파일이나 긴 PDF를 임베드하면 노트가 지나치게 길어지고 로딩 속도가 느려질 수 있습니다.

## 3. 이미지 첨부하고 관리하기
- **다양한 방법으로 이미지 추가하기**:
    - **드래그 앤 드롭**: 컴퓨터의 파일 탐색기에서 이미지 파일을 노트 편집 창으로 끌어다 놓으면 자동으로 추가됩니다.
    - **붙여넣기**: 웹사이트나 클립보드에 복사된 이미지를 `Ctrl+V` (macOS: `Cmd+V`)로 노트에 바로 붙여넣을 수 있습니다.
- **이미지 크기 조절 및 정렬**:
    - **크기 조절**: 이미지 링크 뒤에 `|너비`를 추가하여 크기를 조절할 수 있습니다. 
      예: `![[image.png|300]]`.
    - **정렬**: 이미지 정렬은 사용 중인 테마나 CSS 스니펫을 통해 설정할 수 있습니다.

## 3. PDF 자료 정복하기
- **PDF 파일 추가 및 열람하기**:
    - 이미지를 추가하는 것과 동일하게 PDF 파일을 볼트로 드래그 앤 드롭하여 추가할 수 있습니다.
    - `![[파일이름.pdf]]` 형식으로 임베드하면 노트 내에서 PDF를 바로 보거나, `[[파일이름.pdf]]`로 링크하면 클릭 시 새 탭에서 PDF를 열 수 있습니다.
- **PDF 특정 페이지로 링크하는 방법**:
    - 파일 링크 뒤에 `#page=페이지번호`를 추가하여 특정 페이지로 직접 연결할 수 있습니다.
    - 예시: `[[내논문.pdf#page=5]]`는 5페이지로 링크하고, `![[내논문.pdf#page=10]]`는 10페이지를 노트에 임베드합니다.
- **PDF와 노트를 나란히 보며 작업하기**:
    - `Ctrl` (macOS: `Cmd`) 키를 누른 상태에서 PDF 파일 링크를 클릭하면 새로운 창(Pane)에서 열립니다. 이 기능을 이용해 화면을 분할하여 한쪽에는 PDF를, 다른 한쪽에는 노트를 띄워두고 작업할 수 있습니다.

## 4. 실전 워크플로우: 연구 노트 작성법
- **'리터러쳐 노트' 개념과 작성법**: 리터러쳐 노트란 논문, 책 등 하나의 자료를 읽고 자신의 언어로 핵심 내용을 요약, 정리하는 노트를 말합니다. 옵시디언에서는 각 PDF 자료마다 별도의 노트를 생성하여, 자료의 출처 정보(저자, 연도 등)와 핵심 주장, 인상 깊은 인용구, 자신의 생각 등을 기록하며 관리할 수 있습니다.
- **폴더를 활용한 체계적인 파일 정리**: 자료가 많아지면 폴더 구조를 활용하여 체계적으로 관리하는 것이 좋습니다. 예를 들어, `Sources` 폴더에는 PDF 원본 파일을, `Notes` 폴더에는 리터러쳐 노트를, `Images` 폴더에는 이미지 파일을 모아두는 방식으로 작업 공간을 깔끔하게 유지할 수 있습니다.
- **태그와 속성을 이용한 자료 분류**: 폴더만으로 분류하기 어려운 자료들은 태그와 속성을 활용하면 유연하게 관리할 수 있습니다. `#논문`이나 `#인공지능` 같은 태그로 주제를 표시하거나, 노트 상단에 `author: [저자]`, `year: 2023`과 같은 속성(Properties)을 추가하여 자료를 체계적으로 분류하고 나중에 쉽게 검색하고 필터링할 수 있습니다.
