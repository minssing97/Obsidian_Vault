## 1. 제목 (Headings)
- H1부터 H6까지 제목을 만드는 방법

마크다운에서 제목은 `#` 기호로 만듭니다. `#`의 개수에 따라 제목의 수준이 결정되며, 1개부터 6개까지 사용할 수 있습니다. `#` 다음에 한 칸을 띄고 제목을 입력하세요.

**예시:**
```markdown
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
###### 제목 6
```

## 2. 목록 (Lists)

### 순서 없는 목록 (Unordered Lists)
`*`, `-`, 또는 `+` 기호를 사용하여 순서 없는 목록을 만들 수 있습니다. 기호 뒤에 한 칸을 띄고 내용을 입력하세요.

**예시:**
```markdown
* 항목 1
- 항목 2
+ 항목 3
```

### 순서 있는 목록 (Ordered Lists)
숫자와 점(`.`)을 사용하여 순서 있는 목록을 만듭니다. 숫자를 순서대로 입력하지 않아도 자동으로 번호가 매겨집니다.

**예시:**
```markdown
1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목
```

### 중첩 목록 (Nested Lists)
목록 항목을 들여쓰기(스페이스 2~4칸 또는 탭)하여 중첩된 목록을 만들 수 있습니다.

**예시:**
```markdown
1. 첫 번째 항목
   - 하위 항목 A
   - 하위 항목 B
2. 두 번째 항목
   1. 하위 항목 1
   2. 하위 항목 2
```

## 3. 텍스트 강조 (Emphasis)
텍스트를 강조하는 여러 방법이 있습니다.

-   **굵게**: 별표(`**`)나 밑줄(`__`) 두 개로 텍스트를 감쌉니다.
-   **기울임**: 별표(`*`)나 밑줄(`_`) 한 개로 텍스트를 감쌉니다.
-   **취소선**: 물결표(`~~`) 두 개로 텍스트를 감쌉니다.

`** 이렇게 **` 쓰면 **이렇게** 보인다.
`~~이렇게~~` 쓰면 ~~이렇게~~ 보입니다. 
`*이렇게*` 쓰면 *이렇게* 보입니다.

## 4. 기타 유용한 문법

### 링크 (Links)
대괄호 `[]` 안에 링크 텍스트를, 소괄호 `()` 안에 URL 주소를 입력합니다.
`[표시할 텍스트](https://example.com)` 이렇게 쓰면,
[표시할 텍스트](https://example.com) 이렇게 보이고, 클릭하면 괄호 안의 링크로 이동합니다. 

### 인용문 (Blockquotes)
인용하려는 텍스트 앞에 `>` 기호를 붙입니다. 중첩하여 사용할 수도 있습니다.
```markdown
> 이것은 인용문입니다.
>> 이것은 중첩된 인용문입니다.
```

위와 같이 쓰면 아래와 같이 보입니다.

> 이것은 인용문입니다.
>> 이것은 중첩된 인용문입니다.

---
### 수평선 (Horizontal Rules)
별표(`***`), 하이픈(`---`), 또는 밑줄(`___`)을 3개 이상 연속으로 입력하여 수평선을 만듭니다.
```markdown
---
***
```

### 코드 블록 (Code Blocks)
한 줄 코드는 백틱(`` ` ``)으로 감싸고, 여러 줄로 된 코드 블록은 백틱 3개(```` ``` ````)로 감싸서 만듭니다.
```
	```python
	def hello():
	    print("Hello, world!")
	```
```
