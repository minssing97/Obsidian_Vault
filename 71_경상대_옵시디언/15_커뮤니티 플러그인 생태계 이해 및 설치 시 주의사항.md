## 1. 옵시디언 커뮤니티 플러그인 생태계 이해
- **커뮤니티 플러그인이란 무엇이며, 코어 플러그인과 어떻게 다른지 알아봅니다.**
  - 코어 플러그인: 옵시디언 팀이 직접 개발하고 공식 지원하는 기본 기능
  - 커뮤니티 플러그인: 전 세계 사용자들이 개발하여 공유하는 확장 기능
  - '안전 모드(Safe Mode)'의 역할과 커뮤니티 플러그인 활성화 방법
- **커뮤니티 플러그인 사용의 장점과 잠재적인 단점을 살펴봅니다.**
  - 장점: 무한한 확장성, 개인화된 기능, 생산성 향상
  - 단점: 잠재적인 보안 위험, 버그 또는 충돌 가능성, 개발 중단 위험
- **필요에 맞는 플러그인을 효과적으로 탐색하고 선택하는 기준을 제시합니다.**
  - 옵시디언 내에서 플러그인 검색 및 정렬하는 방법
  - 선택 기준: 다운로드 수, 마지막 업데이트 날짜, GitHub 저장소 활동 확인

---

## 2. 플러그인 설치 및 관리 시 주의사항
- **플러그인 설치 전 반드시 확인해야 할 사항(업데이트 날짜, GitHub 저장소 등)을 알아봅니다.**
  - 개발자 문서(README)를 통해 기능 및 사용법 숙지하기
  - GitHub 'Issues' 탭을 통해 알려진 문제점이나 사용자 피드백 확인하기
  - 최신 옵시디언 버전과 호환되는지 확인하기
- **플러그인 충돌을 방지하고, 안전하게 플러그인을 관리 및 업데이트하는 방법을 배웁니다.**
  - 정기적인 볼트(Vault) 백업의 중요성
  - 플러그인은 한 번에 하나씩 업데이트하며 변경 사항 확인하기
  - 더 이상 사용하지 않는 플러그인은 비활성화 및 삭제하기
- **플러그인으로 인해 문제가 발생했을 때 대처하는 방법을 익힙니다.**
  - 문제 해결 기본 단계: 옵시디언 재시작, 안전 모드 활성화
  - 원인이 되는 플러그인 찾기: 최근에 설치/업데이트한 플러그인부터 순차적으로 비활성화하기
  - 개발자 콘솔(Ctrl+Shift+I)을 통해 오류 메시지 확인하기

## 3. 추천 커뮤니티 플러그인 및 활용 사례
- **생산성과 노트 작성 경험을 향상시키는 필수 플러그인을 소개합니다.**
  - **Dataview**: 노트를 데이터베이스처럼 쿼리하고, 동적 목록을 생성
  - **Templater**: 강력한 템플릿 문법으로 노트 생성을 자동화
  - **Tasks**: 마크다운 파일 내에서 할 일(Task)을 효율적으로 관리
- **특정 목적(지식 관리, 프로젝트 관리 등)에 유용한 플러그인과 활용 사례를 살펴봅니다.**
  - **지식 관리**: Excalidraw (화이트보드/다이어그램), Kanban (칸반 보드)
  - **글쓰기**: Longform (장문 글쓰기 프로젝트 관리), Linter (마크다운 스타일 교정)
  - **꾸미기**: Style Settings (테마 및 플러그인 CSS 맞춤 설정)
