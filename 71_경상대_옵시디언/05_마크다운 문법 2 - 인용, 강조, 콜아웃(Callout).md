## 1. 표 (Tables)
- 데이터를 깔끔하게 정리하는 표 만들기
- 셀 정렬 방법 (왼쪽, 가운데, 오른쪽)

파이프(`|`)와 하이픈(`-`)을 사용하여 표를 만들 수 있습니다. 헤더와 본문을 구분하는 라인에 콜론(`:`)을 추가하여 텍스트 정렬을 제어할 수 있습니다.

-   `:`---`: (기본값) 왼쪽 정렬
-   `:---:`: 가운데 정렬
-   `---:`: 오른쪽 정렬

**예시:**
```markdown
| 제목 1 | 제목 2 | 제목 3 |
| :--- | :---: | ---: |
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| 셀 1-1 | 셀 1-2 | 셀 1-3 |
| 셀 2-1 | 셀 2-2 | 셀 2-3 |
```

| 제목 1  |  제목 2  |   제목 3 |
| :---- | :----: | -----: |
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| 셀 1-1 | 셀 1-2  |  셀 1-3 |
| 셀 2-1 | 셀 2-2  |  셀 2-3 |

## 2. 작업 목록 (Task Lists)
- `[ ]`를 이용한 체크리스트 만들기 및 관리

목록 기호(`-`, `*`, `+`) 뒤에 `[ ]`를 추가하여 작업 목록을 만들 수 있습니다. 대괄호 안에 `x`를 넣으면 완료된 항목으로 표시됩니다. 옵시디언에서는 체크박스를 클릭하여 상태를 변경할 수 있습니다.

```markdown
- [x] 마크다운 기본 문법 공부하기
- [ ] 고급 문법 강의 자료 만들기
- [ ] 옵시디언 플러그인 설치하기
```

위와 같이 코딩하면, 다음과 같이 보이게 됩니다.

- [x] 마크다운 기본 문법 공부하기
- [ ] 고급 문법 강의 자료 만들기
- [ ] 옵시디언 플러그인 설치하기

## 3. 각주 (Footnotes)
본문에 보충 설명이 필요할 때 각주를 사용합니다. 텍스트에 `[^식별자]`를 추가하고, 문서 하단에 `[^식별자]: 내용` 형식으로 설명을 작성합니다. 식별자는 숫자나 문자를 사용할 수 있습니다.

**예시:**
```markdown
옵시디언은 강력한 노트 앱입니다.[^1] 마크다운 문법을 활용하면 생산성을 높일 수 있습니다.[^obs]

[^1]: 옵시디언(Obsidian)은 로컬 기반의 마크다운 파일 관리 도구입니다.
[^obs]: 자세한 내용은 공식 문서를 참고하세요.
```

위와 같이 코딩하면 다음과 같이 보이게 됩니다.

옵시디언은 강력한 노트 앱입니다.[^1] 마크다운 문법을 활용하면 생산성을 높일 수 있습니다.[^obs]

[^1]: 옵시디언(Obsidian)은 로컬 기반의 마크다운 파일 관리 도구입니다.
[^obs]: 자세한 내용은 공식 문서를 참고하세요.

## 4. 콜아웃 (Callouts)
- `[!info]`, `[!warning]` 등으로 텍스트를 강조하는 방법
- 다양한 콜아웃 종류와 커스터마이징

콜아웃은 옵시디언에서 특정 정보를 시각적으로 강조하기 위해 사용하는 기능입니다. 인용문(`>`) 내부에 `[!타입]`을 추가하여 만듭니다.

`note`, `info`, `todo`, `tip`, `success`, `question`, `warning`, `error` 등 다양한 타입을 지원하며, `[!타입] 제목` 형식으로 사용자 정의 제목을 추가할 수도 있습니다.

**예시:**

```markdown
> [!note]
> 이것은 노트 콜아웃입니다. 추가적인 정보를 기록할 때 사용합니다.

> [!warning] 중요 공지
> 이 설정은 앱 성능에 영향을 줄 수 있습니다.
```

> [!note]
> 이것은 노트 콜아웃입니다. 추가적인 정보를 기록할 때 사용합니다.

> [!warning] 중요 공지
> 이 설정은 앱 성능에 영향을 줄 수 있습니다.

## 5. 노트 임베딩 (Embedding Notes)
- 다른 노트의 전체 내용이나 특정 문단을 현재 노트에 삽입하는 방법 (Transclusion)

옵시디언에서는 `![[노트이름]]` 형식을 사용하여 다른 노트의 내용을 현재 노트에 그대로 가져와 표시할 수 있습니다. 이를 '임베딩' 또는 '트랜스클루전'이라고 합니다.

-   **노트 전체 임베딩**: `![[가져올 노트 제목]]`
-   **특정 제목 아래 내용 임베딩**: `![[노트 제목#제목]]`
-   **특정 블록(문단) 임베딩**: `![[노트 제목#^블록ID]]` (블록 ID는 문단 끝에 `^원하는ID`를 추가하여 만듭니다.)

**예시:**
```markdown
다른 노트의 내용을 여기에 삽입합니다.

![[테스트를 위한 임시 노트입니다.]]

특정 제목 아래 내용만 가져올 수도 있습니다.

![[테스트를 위한 임시 노트입니다.#이것은 두 번째 제목입니다.]]
```

위와 같이 코딩하면, 다음과 같이 문서 내용 자체가 현재 문서 내에 Embedding 되는 모습을 확인할 수 있습니다.

다른 노트의 내용을 여기에 삽입합니다.

![[테스트를 위한 임시 노트입니다.]]

특정 제목 아래 내용만 가져올 수도 있습니다.

![[테스트를 위한 임시 노트입니다.#이것은 두 번째 제목입니다.]]

## 6. 기타 유용한 문법
- 주석(Comments)을 이용해 메모 남기기
- HTML 태그를 활용한 고급 서식 지정

### 주석 (Comments)
`%%`로 텍스트를 감싸면 주석으로 처리됩니다. 주석은 편집 모드에서만 보이고 읽기 모드에서는 표시되지 않아, 메모를 남기는 용도로 활용할 수 있습니다.

**예시:**
```markdown
이 내용은 화면에 보입니다.
%%이 내용은 주석이라 화면에 보이지 않습니다.%%
```

### HTML 태그 활용
마크다운은 HTML 태그를 지원하므로, 밑줄(`<u>`), 접고 펼 수 있는 상세 정보(`<details>`) 등 더 다양한 서식을 적용할 수 있습니다.

**예시:**
```markdown
<u>이 텍스트에는 밑줄이 그어집니다.</u>

<details>
<summary>여기를 클릭하여 상세 내용 보기</summary>
숨겨져 있던 상세 내용이 여기에 나타납니다.
</details>
```

<u>이 텍스트에는 밑줄이 그어집니다.</u>

<details>
<summary>여기를 클릭하여 상세 내용 보기</summary>
숨겨져 있던 상세 내용이 여기에 나타납니다.
</details>
