이 문서는 옵시디언(Obsidian)을 처음 설치하고 가장 효율적인 사용 환경을 구축하기 위한 단계별 가이드입니다.

## 1단계: 옵시디언 프로그램 다운로드 및 설치

옵시디언은 Windows, macOS, Linux, iOS, Android 등 모든 플랫폼에서 사용할 수 있습니다.

1. **공식 웹사이트 접속:** 웹 브라우저를 열고 `obsidian.md`로 이동합니다.    
2. **다운로드:** 메인 화면의 'Download' 버튼을 클릭하여 사용하는 운영체제에 맞는 설치 파일을 선택해 다운로드합니다.    
3. **실행:** 다운로드한 설치 파일을 실행하고 안내에 따라 설치를 완료합니다.
    - **모바일 사용자:** App Store(iOS) 또는 Play Store(Android)에서 "Obsidian"을 검색하여 설치할 수 있습니다.
          

---

## 2단계: 첫 번째 볼트(Vault) 생성하기

**볼트(Vault)**는 옵시디언 노트 파일(`.md` 파일)과 설정을 모두 저장하는 마스터 폴더입니다.

1. **옵시디언 실행:** 설치 후 프로그램을 처음 실행합니다.    
2. **볼트 생성 선택:** 시작 화면에서 **"새 볼트 생성 (Create new vault)"**을 클릭합니다.    
3. **이름 및 위치 지정:**
    - **볼트 이름:** 사용할 이름을 입력합니다. (예: `나의 지식 창고`)
    - **위치 지정:** **'찾아보기 (Browse)'** 버튼을 클릭하여 볼트가 생성될 폴더 경로를 지정합니다.
        - **💡 팁:** Google Drive, iCloud Drive, Dropbox 등 **클라우드 서비스가 동기화되는 폴더** 안에 볼트를 생성하면, 자동으로 모바일과 PC 간 동기화가 이루어집니다.
4. **생성:** **"생성 (Create)"** 버튼을 클릭하여 첫 번째 볼트를 엽니다.

## 3단계: 필수 초기 설정 및 단축키 지정
옵시디언의 효율을 극대화하기 위해 초기 환경을 설정합니다.

### 3.1. 설정 메뉴 진입
- **방법 1 (추천):** `Ctrl` + `P` (Mac은 `Cmd` + `P`)를 눌러 **명령어 팔레트**를 연 다음, "Open Settings"를 입력하고 엔터키를 누릅니다.    
- **방법 2:** 앱 좌측 하단 구석에 있는 **톱니바퀴 아이콘 (설정 버튼)**을 클릭합니다.

### 3.2. 파일 및 링크 설정 (Settings > Files & Links)
노트와 첨부 파일의 저장 경로를 분리하여 깔끔하게 관리합니다.

| **설정 항목**                                                   | **권장 설정**                                                    | **활용법**                                                 |
| ----------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------- |
| **새 노트 기본 위치**<br>(Default Location For New Notes)          | **In the folder specified below** 선택 후 `00-Inbox` 폴더 지정      | 생각나는 즉시 메모하고 나중에 정리하기 편리합니다.                            |
| **새 첨부 파일 기본 위치**<br>(Default Location for New Attachments) | **In the folder specified below** 선택 후 `99-Attachment` 폴더 지정 | 노트(`.md`) 파일과 이미지, PDF 등 첨부 파일을 분리하여 파일 목록을 깔끔하게 유지합니다. |
| **WikiLinks 사용**<br>(use [[Wikilinks]])                     | **활성화 (Enable)**                                             | 옵시디언의 핵심 연결 기능입니다.                                      |
| **내부 링크 자동 업데이트**<br>(Automatically Update Internal Links)  | **활성화 (Enable)**                                             | 노트 이름을 변경해도 링크가 깨지지 않고 자동으로 수정됩니다.                      |

### 3.3. 단축키(Hotkeys) 커스터마이징 (Settings > Hotkeys)

가장 자주 사용하는 내부 링크 생성 기능을 단축키로 지정합니다.

1. **메뉴 이동:** `설정 > Hotkeys` 메뉴로 이동합니다.
2. **기능 검색:** 검색창에 **`Add Internal Link`**를 입력합니다.
3. **단축키 지정:** 해당 기능 옆의 '추가' 버튼을 클릭하고, **`Ctrl` + `[`** (Mac은 `Cmd` + `[`)를 눌러 단축키를 설정합니다.

|                              | **기능**                           | **설명**                                  |
| ---------------------------- | -------------------------------- | --------------------------------------- |
| `Ctrl`/`Cmd` + `[`           | **내부 링크 추가 (Add Internal Link)** | 선택된 텍스트를 즉시 `[[링크]]`로 변환하여 새 문서를 연결합니다. |
| `Ctrl`/`Cmd` + `O`           | **빠른 전환 (Quick Switcher)**       | 모든 노트를 검색하여 빠르게 이동합니다.                  |
| `Ctrl`/`Cmd` + `Shift` + `F` | **전체 검색 (Search)**               | 볼트 내 모든 노트의 내용을 검색합니다.                  |

## 4단계: 핵심 코어 플러그인 및 테마 설정
### 4.1. 코어 플러그인 활성화 (Settings > Core Plugins)
옵시디언이 기본 제공하는 강력한 내장 기능들입니다. 다음 플러그인들은 대부분의 사용자에게 유용하므로 활성화를 추천합니다.

| 플러그인                | 설명                             | 추천 이유                                        |
| ------------------- | ------------------------------ | -------------------------------------------- |
| **Templates**       | 노트 템플릿을 생성하고 사용합니다.            | 반복적인 노트 형식(회의록, 독서 노트 등)을 저장하여 생산성을 높여줍니다.   |
| **Daily notes**     | 오늘 날짜의 노트를 빠르게 생성합니다.          | 매일의 생각, 업무, 일기를 기록하는 습관을 만드는 데 핵심적인 역할을 합니다. |
| **Command palette** | `Ctrl/Cmd+P`로 명령어를 검색하고 실행합니다. | 옵시디언의 모든 기능에 빠르게 접근할 수 있어 작업 속도를 크게 향상시킵니다.  |
| **Slash commands**  | `/`를 입력하여 다양한 명령을 실행합니다.       | 글을 쓰는 흐름을 끊지 않고 템플릿 삽입 등을 쉽게 사용할 수 있습니다.     |
| **Page preview**    | 링크에 마우스를 올리면 노트 내용을 미리 봅니다.    | 링크를 클릭하지 않고도 내용을 빠르게 확인할 수 있어 탐색이 매우 편리해집니다. |
| **File recovery**   | 주기적으로 스냅샷을 저장하여 데이터 복구를 돕습니다.  | 실수로 데이터를 삭제했을 때를 대비한 중요한 안전장치입니다.            |

## 5단계: 모바일/기기 간 동기화 방법
### 5.1. 옵시디언 싱크 (Obsidian Sync - 유료)
가장 간단하고 안전한 방법으로, 옵시디언에서 제공하는 공식 유료 서비스입니다.

- **장점:** 복잡한 설정 없이 로그인만으로 동기화 가능, End-to-End 암호화(보안), 노트 버전 관리(복구 기능).
- **설정:** `설정 > Sync` 메뉴에서 계정에 로그인하고 동기화를 설정합니다.
    
### 5.2. 무료 클라우드 동기화 (iCloud, Drive 등)
볼트(폴더)를 클라우드 저장소에 저장하여 동기화합니다.

|                             |                                                  |                                                            |
| --------------------------- | ------------------------------------------------ | ---------------------------------------------------------- |
| **클라우드 서비스**                | **특징**                                           | **모바일 지원**                                                 |
| **iCloud Drive**            | Apple 기기(Mac, iPhone, iPad) 사용자에게 가장 추천되는 무료 방법. | 모바일 앱이 볼트 폴더에 직접 접근 가능하여 완벽하게 호환됩니다.                       |
| **Google Drive / OneDrive** | PC 간 동기화에 용이합니다.                                 | iOS/Android 모바일 앱에서 동기화를 위해 **별도의 커뮤니티 플러그인**이 필요할 수 있습니다. |
