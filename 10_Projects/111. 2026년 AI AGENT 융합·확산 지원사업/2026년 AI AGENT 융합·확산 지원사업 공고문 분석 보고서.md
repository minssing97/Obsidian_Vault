# 

> **공고번호** : 과학기술정보통신부 공고 제2026-0222호  
> **공고일** : 2026. 2. 27.  
> **전담기관** : 정보통신산업진흥원(NIPA)

---

## 1. 사업 개요

| 항목 | 내용 |
|------|------|
| **사업명** | 2026년 AI AGENT 융합·확산 지원 |
| **주무부처** | 과학기술정보통신부 |
| **전담기관** | 정보통신산업진흥원(NIPA) |
| **사업목적** | 산업 분야 특화 AI 에이전트 서비스 개발·실증으로 국민 편익 증대 및 AI산업 경쟁력 강화 |
| **지원규모** | 총 8개 과제 / '26년 총 92.8억원 / 과제당 11.6억원 이내 |
| **사업기간** | 협약체결일 ~ 2027.11.30. (2개년) |
| **접수마감** | **2026. 3. 30.(월) 14:00** (전산접수 시작: 3.16.(월) 10:00) |
| **접수방법** | NIPA 사업관리시스템([https://nxt.nipa.kr](https://nxt.nipa.kr)) 온라인 접수 |

### 추진 배경
- 생성형 AI → **AI 에이전트 기반 서비스**로 기술 패러다임 전환 중
- AI 에이전트 글로벌 시장 전망: ('23) 35억달러 → ('30년e) **471억달러**
- 국내는 기술역량 대비 산업 현장 적용 AI 에이전트 사례·실증 경험 제한적
- 정부 주도 서비스 개발·초기 실증 지원을 통한 성공 사례 조기 창출 필요

## 3. 신청 자격 요건

### 기본 요건
- **법인사업자만 신청 가능** (개인사업자 불가)
- AI 에이전트 서비스 개발이 가능한 **중소·중견 기업**
- **스타트업**: 접수 마감일까지 설립 완료된 기업 (예정 기업 불가)
- 상용 LLM 모델 사용 불가 → **오픈소스 기반 개발 필수**

### 컨소시엄 구성 (필수)

| 역할 | 자격 | 수 |
|------|------|---|
| **주관기관** | AI 에이전트 서비스 제공 가능한 중소·중견기업 | 1개 (필수) |
| **수요기관** | 데이터 제공 및 실증 가능 기업 (중소·중견·대기업) | 1개 이상 (필수) |
| 참여기관 (AI기업 등) | AI 에이전트 서비스 제공 기업 | 필요 시 (최대 4개) |

> [!IMPORTANT]
> - 1개 기업은 1개 컨소시엄에만 참여 가능 (복수 참여 불가)
> - 컨소시엄 구성원 상호간 계열사 참여 불가
> - 공공기관, 지방자치단체, 중앙행정기관은 참여기관으로 불가

## 7. 사업계획서 목차 구성

사업계획서(붙임 3 양식)는 다음 8개 장으로 구성됩니다:

```
Ⅰ.  사업 개요          (추진배경, 필요성, 사업범위)
Ⅱ.  산업동향 및 현황 분석 (국내외 동향, 기술현황, 제약요인, 기대효과)
Ⅲ.  추진 목표 및 전략   (최종목표, 추진방향, 추진체계, 컨소시엄 역할)
Ⅳ.  세부 추진계획       (목표서비스, 세부과제, 장비, 성능확보, 일자리, 파급효과)
Ⅴ.  사업 관리계획       (산출물, 보고계획, 추진일정)
Ⅵ.  기관별 수행 상세내역 (주관/참여/용역 계획서)
Ⅶ.  주요 사업추진 실적 및 참여인력 현황
Ⅷ.  사업비 소요명세     (총괄표, 기관별 세부현황)
```

## 8. 사업계획서 작성을 위해 사전에 준비해야 할 항목

### 📌 [기업/컨소시엄 구성 단계]

- [ ] **주관기업 확정**: 중소/중견기업 법인, AI 에이전트 개발 역량 보유 여부 확인
- [ ] **수요기업 발굴**: 실증 데이터를 제공할 수 있는 파트너 기업 MOU 체결
- [ ] **참여기업 구성**: 필요시 AI 서비스 개발 참여기업 추가 (최대 4개)
- [ ] **컨소시엄 계획서**: 기관별 R&R(역할·책임) 구체화 및 상호계약 준비
- [ ] **Track 결정**: Track1(핀테크 or 뷰티테크) vs Track2(자율제안) 선택

### 📌 [서비스 기획 단계]

- [ ] **AI 에이전트 서비스 아이디어**: 특정 산업 도메인에 특화된 문제 해결 시나리오 수립
  - 인사·회계·ERP 등 범용 내부업무 지원 서비스는 **지원 대상 제외**
  - 민간 분야 서비스에 한함 (공공분야 제외)
- [ ] **다중 에이전트 아키텍처 설계**: 목표→계획→실행→관찰→수정의 반복 구조
  - **에이전트 수 5종 이상** 설계 필요 (필수 목표 지표)
- [ ] **오픈소스 LLM 선정**: 상용 LLM 사용 불가, 오픈소스 기반 모델 결정 (근거 포함)
- [ ] **NPU 활용 계획**: AI 솔루션 서비스 제공을 위한 NPU 활용 여부 검토
- [ ] **서비스 시나리오 수립**: 수요기업 환경에서의 실증 시나리오 최소 3종 이상

### 📌 [데이터 준비 단계]

- [ ] **학습 데이터 구축 계획**: 수요기업 보유 데이터 현황 파악 및 수집 계획 수립
  - Trajectory 데이터 10,000건 이상 (1차년도 필수 목표)
  - 예외 대응 데이터 1,000건 이상
- [ ] **데이터 활용 법적 검토**: 개인정보보호법 준수 방안 수립
- [ ] **인공지능 개인정보보호 자율점검**: 기획·설계·수집·이용·보관·파기 단계별 점검표 작성
- [ ] **데이터 중복지원 확인**: NTIS 등 타 정부과제와 데이터 중복 여부 사전 확인

### 📌 [성과 목표 설정 단계]

성과 지표 목표치 (1차년도 기준):

| 지표 | 목표치 |
|------|--------|
| 신규 고용창출 | 7명 이상 |
| AI 에이전트 수 | 5종 이상 |
| 협업 워크플로 유형 | 3종 이상 |
| 응답 정확도 | 85% 이상 |
| 응답 일관성 | 90% 이상 |
| 환각 발생률 | 5% 이하 (치명적 오류 0%) |
| 사용자 만족도 | 60점 이상 |
| 사업화 성과 | MOU 2건 이상 |

- [ ] **정량/정성 목표 수립**: 위 지표 기준으로 달성 가능한 수치 설계
- [ ] **공인 검증기관 확보**: KOLAS(한국인정기구) 인증기관과 사전 협의
- [ ] **일자리 창출 계획 수립**: 신규채용 계획 + 4대 보험 가입률 계획

### 📌 [사업비 편성 단계]

- [ ] **사업비 규모 결정**: 정부지원금(최대 11.6억) + 민간매칭 구성
- [ ] **민간매칭 확보**: 중소기업 기준 총 사업비의 25% 이상 (현금 최소 2.5%)
- [ ] **사업비 비목별 편성**:
  - 인건비 (참여인력 급여 대장 필수)
  - 운영비 (회계정산 수수료 필수 포함, 성능 검증 수수료 포함)
  - 여비 (국내/해외 출장)
  - 업무추진비 (총 사업비의 2% 이내)
  - 자산취득비 (총 사업비의 10% 이내, PC·노트북 등 범용장비 신규인력만 가능)
- [ ] **위탁정산 수수료 계상**: 사업비 규모에 따라 반드시 편성 (아래 기준)
  - 5억~10억: 1,600천원 / 10억~30억: 1,800천원
- [ ] **별도 계좌 준비**: 사업비(정부지원금+민간현금)를 관리할 별도 계좌
- [ ] **지급이행보증보험**: 협약 시 제출 필요 (사전에 보험사 협의)

### 📌 [제출 서류 준비 단계]

- [ ] **사업자등록증 or 법인등기부등본** (공고일 이후 발급)
- [ ] **표준재무제표증명원** (2023~2025년, 3개년) - '25년 미신고 시 결산재무제표+세무대리인 직인
- [ ] **국세·지방세 완납증명서** + **4대 보험 완납증명서**
- [ ] **기관확인서** (중소기업/중견기업 구분 확인서)
- [ ] **NTIS 유사과제 검색 결과** (차별성검토 40점 설정 후 출력)
- [ ] **일자리 창출계획서** (엑셀 양식 활용)
- [ ] **자산취득 활용계획서** (3천만원 이상 장비 해당 시)
- [ ] **지급이행보증보험증권** (선정 후 협약 시)

### 📌 [사전 자격 확인]

- [ ] **참여제한 여부 확인**: NTIS(ntis.go.kr) → 사업관리 → 제재정보에서 기관명/이름 검색
- [ ] **복수과제 중복지원 여부**: 1개 기업은 1개 컨소시엄에만 가능
- [ ] **재무 건전성 체크**: 자본전액잠식, 부도, 세금 체납 여부 확인
- [ ] **인건비 계상률**: 참여인력별 총 계상률 100% 초과 여부 점검
- [ ] **과거 개인정보법 위반 이력**: 있을 경우 평가 시 3점 감점
