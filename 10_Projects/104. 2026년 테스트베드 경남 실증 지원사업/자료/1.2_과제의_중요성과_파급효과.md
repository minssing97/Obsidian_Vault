### 과제의 중요성과 파급효과

#### 과제의 중요성

##### 실시간성 부족(지연/누락) 문제

○ 현재 상태: MCT 현장의 기존 모니터링 환경은 Controller 파라미터(RPM·Feed·Load 등)를 수치 화면으로만 표시하며, 데이터 수집–전송 과정에서 지연·누락이 발생하더라도 이를 즉시 인지하기 어려운 구조임
- 발생 리스크: 알람 발생 후 운영자가 인지하기까지 평균 수십 초~수 분이 소요될 수 있으며, 이 지연은 가공 불량(품질 저하), 장비 정지(가동률 하락), 안전사고(충돌·과부하)로 직결됨. 공작기계 비계획 정지 1회당 손실은 수백만 원 규모로 추산됨
- 본 과제 해결요소: Controller → Collector → Middleware → Unity 경로에 시퀀스 ID 기반 누락 탐지, 버퍼/재전송, keep-alive 자동 재연결 로직을 적용하여 end-to-end 지연시간 P95 0.5초 이내, 데이터 누락률 0.1% 이하, 재연결 복구 10초 이내를 정량 목표로 관리함

##### 2D 수치 중심 모니터링 한계

○ 현재 상태: RPM·Feed·Load 등 수치가 화면에 표시되더라도, 현재 어느 축이 움직이고 있는지·주축 부하가 어떤 동작 맥락에서 상승하는지를 운영자가 직관적으로 판단하기 어려움
- 발생 리스크: 상황판단 지연으로 인해 알람 원인 파악·조치 의사결정이 늦어지며, 특히 교대근무·다기종 운영 환경에서 숙련도 편차가 품질 산포로 이어짐
- 본 과제 해결요소: Unity 3D 모델에 Controller 파라미터를 매핑(RPM→주축 회전 애니메이션, Feed→축 이동속도, Load→부하 게이지 색상 경보)하여 장비 동작을 실시간 3D로 재현함. 상태머신(Idle/Run/Alarm) 기반 전이 가시화를 통해 상황판단 시간을 단축하고 의사결정을 지원함

##### 교육·훈련 제약(안전/비용/가동중단)

○ 현재 상태: MCT 실습 교육은 실제 장비를 사용해야 하며, 교육 중 장비 가동을 중단해야 하므로 생산 손실이 발생함. 제조 현장 교육훈련 시간은 월 평균 8~16시간으로 제한적이며, 안전사고 위험으로 반복 훈련이 어려움
- 발생 리스크: 숙련 기술자 고령화·퇴직에 따른 기술 전수 단절, 신규 인력의 장비 조작 미숙으로 인한 가공 불량·공구 파손·안전사고 발생 가능성 증가
- 본 과제 해결요소: Unity 디지털트윈 기반 가상 실습 환경을 구축하여 장비 가동 중단 없이 주축 회전·이송·부하·알람 상태를 3D로 재현함. 공동수행기관(한국폴리텍대학)과 연계하여 표준 작업 시나리오 기반 교육 콘텐츠를 개발하며, 교육 접근성과 안전성을 동시에 개선함

##### 기종/프로토콜 종속 및 확장 비용

○ 현재 상태: Controller 제조사(Fanuc·Siemens·Mitsubishi 등)별로 통신 프로토콜과 파라미터 구조가 상이하여, 기종이 변경될 때마다 연동 모듈을 별도로 개발해야 함. 국내 중소 제조기업의 공작기계 보유 대수는 사업장당 평균 5~10대이나, 다기종 혼용 비율이 높아 통합 관리가 어려운 실정임
- 발생 리스크: 기종 추가 시 개발·유지보수 비용이 선형적으로 증가하며, 데이터 형식 불일치로 인해 라인 단위·공장 단위 통합 관제가 사실상 불가능함
- 본 과제 해결요소: 데이터사전(태그명·단위·주기·매핑 규칙)과 매핑 템플릿을 표준화하여, 신규 기종 추가 시 템플릿 교체만으로 연동할 수 있는 확장 구조를 설계함. 이를 통해 단일 장비 실증에서 다장비·라인 확장으로의 전환 비용을 절감함

##### 실증 이후 사업화 연결 필요

○ 현재 상태: 정부 R&D 실증 과제는 기술 검증으로 종료되는 경우가 많아, 현장 도입·매출·확산으로 이어지지 못하는 구조적 문제가 있음
- 발생 리스크: 실증 결과물이 1회성 시제품에 머물 경우, 투입된 공적 자금 대비 산업적 성과(매출·고용·기술 확산)가 미흡하게 됨
- 본 과제 해결요소: 실증 단계부터 유상 전환/계약·증빙 체계를 설계하여 사업화율 130% 이상(목표 매출 1.56억원 / 과제 지원금 1.2억원)을 달성 목표로 설정함. 신규채용 2명·순고용증가 2명을 통해 3D 모델링·Unity 개발·현장 데이터 엔지니어링 직무 인력을 확보하고, 경남 지역 디지털 제조 인력 기반을 강화함

#### 과제의 파급효과

##### 기술적 측면

○ Controller 데이터 정규화/데이터사전
- 현재 한계: Controller 제조사별로 태그명·단위·주기가 상이하여 데이터 활용·비교·통합이 어려움
- 과제 개선: 태그명·단위·수집주기·매핑 규칙을 통일한 데이터사전(v1.0)을 구축하여, 파라미터별 의미·범위·임계값을 표준 정의함
- 확장 분야: 타 기종(선반·연삭기·방전가공기 등) 및 타 시스템(MES·ERP·클라우드)과의 데이터 연계 기반을 마련하며, 제조 데이터 표준화 레퍼런스로 활용 가능함

○ 실시간 스트리밍/동기화 품질
- 현재 한계: 기존 모니터링은 폴링(polling) 방식 중심으로 지연이 크고, 네트워크 단절 시 데이터 유실이 발생함
- 과제 개선: 시퀀스 ID 기반 누락 탐지, 버퍼/재전송, keep-alive 자동 재연결 로직을 미들웨어에 구현하여 지연 P95 0.5초 이내·누락률 0.1% 이하·복구 10초 이내를 달성함
- 확장 분야: 실시간 관제·원격 모니터링·Edge-Cloud 하이브리드 아키텍처 등 산업용 스트리밍 기술 전반에 파급 가능함

○ Unity 상태머신 기반 동작 재현
- 현재 한계: 3D 시각화는 오프라인 시뮬레이션(CAM 검증 등)에 한정되어 있으며, 실시간 장비 상태와 동기화된 3D 재현 사례는 중소기업 현장에서 극히 제한적임
- 과제 개선: Run/Idle/Alarm 상태머신 전이와 RPM·Feed·Load 파라미터 매핑을 Unity에서 실시간 구현하여, 장비 동작을 3D로 직관 재현함
- 확장 분야: 교육·훈련용 시뮬레이터, VR/AR 기반 원격 지원, 멀티 장비 3D 관제 대시보드 등으로 기술 파급이 가능함

○ 이벤트/알람/로그 체계 및 검증 체계
- 현재 한계: 알람 이력이 Controller 내부에만 잔류하거나, 외부 시스템과 연동되지 않아 추적성·분석성이 부족함
- 과제 개선: 알람 발생/해제·상태 전이·파라미터 변화를 타임스탬프 기반 로그로 체계화하고, 7개 성능 KPI(지연·누락·복구·일치율·매핑정확도·연속운전·로그완전성)에 대한 측정방법·산식·증빙을 포함한 시험성적서를 작성함
- 확장 분야: OEE(설비종합효율) 분석, 예지보전(PdM), 이상징후 탐지 등 데이터 기반 후속 고도화의 기초 인프라로 연계됨

○ 지식재산권 1건 확보
- 미들웨어의 품질 관리 로직(누락 탐지·버퍼/재전송·재연결), 데이터사전 기반 매핑 검증·자동 복구 방법 등을 핵심 청구항으로 하여 특허 또는 프로그램 저작권을 출원함
- 기술적 성과를 IP 자산으로 전환하여 후속 사업화·기술이전·라이선싱의 기반을 확보함

수요기업(중소 제조)	낮은 도입 비용으로 설비 가시화·모니터링 고도화, 스마트공장 기초→중간 단계 전환

○ 공통목표 연계 확산 활동
- 경남 디지털위크 참가 1건: 디지털위크 행사에 전시 부스를 운영하여 Unity Twin 실시간 데모·브로슈어·실증 영상을 공개하고, 방문 수요기업과 리드 미팅을 통해 후속 도입처를 발굴함
- 공동수행기관 협업 네트워크 1건: 한국폴리텍대학과의 산학 공동수행을 통해 네트워킹 행사(세미나·워크숍)를 개최하고, 지역 제조기업·SI·장비업체와의 협업 네트워크를 구축하여 후속 수요 발굴 및 기술 확산의 거점 역할을 수행함

○ 정책 연계
- 과학기술정보통신부 「지역 디지털 혁신거점 조성 지원사업」의 디지털 제조혁신 목표에 직접 부합하며, 스마트제조 혁신 전략의 '데이터 기반 공정 고도화', 제조 DX 정책의 '중소기업 디지털 전환 촉진', 지역 인력양성 정책의 '디지털 제조 전문인력 육성'과 연계하여 정책 목표 달성에 기여함

##### 사회적 측면

○ 안전 향상
- MCT 가공 현장의 위험 작업(고속 회전·절삭·중량물 이송 등)에 대한 교육·훈련을 Unity 디지털트윈 기반 가상 실습으로 대체하여, 교육 중 안전사고 발생 위험을 원천적으로 차단함
- 알람·이상 상태 발생 시 3D 화면에서 해당 부위를 즉시 하이라이트하여, 운영자의 위험 인지 속도를 높이고 신속한 대피·조치 판단을 지원함

○ 숙련 전수 및 청년 인력 유입
- 숙련 기술자의 장비 운전 노하우를 디지털트윈 기반 표준 작업 시나리오(정상 운전·알람 대응·셋업 절차 등)로 문서화·시각화하여, 경험 의존적 숙련 전수를 체계적 교육으로 전환함
- 공동수행기관(한국폴리텍대학)의 교육 인프라를 활용하여 청년 인력 대상 디지털 제조 실무 교육을 실시하고, 신규채용 2명(3D 모델러·Unity 개발자)을 통해 지역 청년 고용에 직접 기여함
- 컨설팅 1건: 기술·경영·마케팅 분야 전문가 자문을 통해 기업의 사업화 역량을 강화하며, 컨설팅 결과보고서를 증빙으로 제출함

○ 삼중효과
- 본 과제는 실시간 모니터링·3D 가시화를 통한 현장 생산성 향상, Twin 기반 가상 실습을 통한 교육 접근성 개선, 위험 작업 대체 및 알람 조기 인지를 통한 안전성 강화를 동시에 달성하여, 제조 현장의 생산성·교육·안전 삼중효과를 실현함
