# 과제개요
Controller의 파라미터값을 입력받아 MCT(Machining Center)의 주축 회전수, Feed, 주축 부하 등 작동 상태를 모니터링할 수 있는 환경이 구축되어 있는 상황에서 MCT의 작동 상황을 Unity를 활용하여 Digital Twin을 구현하는 실증 사업임
## 수행기간 : 8개월(2026.3~2026.10)

## 사업비 : 1.2억원

## 주요 연구/실행 항목
- 실증 사업 대상인 MCT 모델의 3D 모델링
- Unity에서 MCT 3D 형상 구현
- MCT controller parameter 값에 따라 Unity 상에서 MCT 작동 구현
- Controller parameter 값을 실시간으로 Unity에 전달하는 환경 구축

## 연구주체
- 주관기관(알파에듀)
	-  실증 사업 대상인 MCT 모델의 3D 모델링
	- Unity에서 MCT 3D 형상 구현
	- MCT controller parameter 값에 따라 Unity 상에서 MCT 작동 구현
	- Controller parameter 값을 실시간으로 Unity에 전달하는 환경 구축
- 공동연구기관(한국폴리텍대학)
	- Controller의 파라미터값을 입력받아 MCT의 작동 Monitoring
	- Controller의 파라미터값을 Unity에 전달
